Resumo: Para realizar o jogo pertendido foi necessário criar duas modalidades diferentes.
A primeira é quando é o computador a advinhar o número do utilizador. Para isso determinei o range dos numeros que o computador pode escolher ( entre 0 e 100) e enquanto a resposta for diferente de acertou temos duas opçoes dependendo da resposta do utilizador. No fim o computador entrega o numero correta e o numero de tentativas que demorou a acertar
Para a segunda modalidade o computador tem de escolher um numero atraves da operção import random. Enquanto a resposta for diferente do numero escolhido inicialmente o programa diz nos se o input do utilizador é maior ou menor que o numero. 
Por fim realizei um simples menu para o utilizador escolher o modo de jogo que pretende jogar
