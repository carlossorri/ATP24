Temos ainda outras duas vertentes que vao ser iguais independente do modo de jogo escolhindo. 
Na jogada do jogador, peço ao utilizador para selecionar entre 1 a 4 fosforos e se a resposta nao estiver neste intervalo a resposta nao é valida.
Na jofada do computador utilizei uma tecnica para que o numero de fosforos que ficaria disponiveis para o utilziador fosse sempre um numero multiplo de 5. Se isto nao fosse possivel para escolher de forma random um numero de 1 a 4. 
Sempre que o utilizador ou o computador escolha o numero de fosforos a tirar é tirado desse montante ate que chegue a 0. Para isso adicionei a variavel jogo_ativo= True. quando o numero de fosfors=< 0 esta condicao passa a false
